优先级 + 类型
  1. 1. crash — ATR% ≥ 5% → 任何方向都强制 neutral
  2. 2. uptrend — MA20 vs MA120 偏离 ≥ +3%
  3. 3. downtrend — MA20 vs MA120 偏离 ≤ -3%
  4. 4. range_bound — 其他
  5. 5. unknown — 数据不足 → 维持原计划
CIO Sanity Check（防 LLM 幻觉）
  • confidence≥0.95 + verdict=BUY → 自动降级到 ACCUMULATE(0.6)
  • |SUGGESTED_ALLOC_CNY| > 100,000 → clamp 到 ±100,000
  • 任何角色输出异常（CLI 失败/超时/空输出）→ 系统强制 HOLD + confidence ≤ 0.4
  • Risk=high_risk → 即使 Quant+Macro bullish 也最多 ACCUMULATE，禁 BUY
  • CONCENTRATION > 60% → 任何加仓 ≤ dry_powder × 10%
